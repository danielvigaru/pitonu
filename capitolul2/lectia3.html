Tipurile de date din Python

În programare, tipul de date este un concept important.

Variabilele pot stoca date de diferite tipuri, iar diferitele tipuri pot face lucruri diferite.

Python are următoarele tipuri de date încorporate:
  • Tipuri text: str
  • Tipuri numerice: int, float, complex
  • Tipuri secvențiale: list, tuple, range
  • Tipuri mapare: dict
  • Tipuri set: set, frozenset
  • Tipuri booleene: bool
  • Tipuri binare: bytes, bytearray, memoryview
