Ce poate face Python?

Python este un limbaj de programare foarte versatil. Datorită faptului că este open-source, comunitatea lucrează mereu la dezvoltarea diverselor librarii cu ajutorul cărora se pot realiza diferite aplicații.

Astfel, acest limbaj puternic poate realiza urmatoarele operațiuni:

  • Python poate fi folosit pe un server pentru a crea aplicații web.
  • Python se poate conecta la sistemele de baze de date. Poate citi si modifica fișierele.
  • Python poate fi folosit pentru prototipuri rapide sau pentru dezvoltarea software-ului pregătit pentru producție.
  • Python poate fi folosit alături de software pentru a crea workflows (fluxuri de lucru).
  • Python poate crea aplicații desktop, cu librării GUI predefinite.
  • Python poate automatiza foarte multe procese care implică lucratul cu date(fisiere, formulare, etc).
