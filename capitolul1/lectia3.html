De ce Python?

Popularitatea în creștere, dar și puterea limbajului de programare Python au dus la adoptarea sa ca limbaj principal de dezvoltare de către programatori specializați și chiar și la predarea limbajului în unele medii universitare. Din aceleași motive, multe sisteme bazate pe Unix, inclusiv Linux, BSD și Mac OS X includ din start interpretatorul CPython.

Cateva avantaje ale limbajului Python, față de restul limbajelor de programare sunt:
  • Python lucrează pe diferite platforme (Windows, Mac, Linux, etc.)
  • Python are o sintaxă simplă, similară cu limba engleză.
  • Python are o sintaxă care permite dezvoltatorilor să scrie programe cu mai puține linii de cod decât alte limbaje de programare.
  • Python poate fi tratat intr-un mod procedural, orientat pe obiecte (OOP) sau functional.
  • Python este un foarte bun limbaj pentru începători
  • Python este foarte bine documentat, datorită comunității mari de programatori ce lucrează în acest limbaj
